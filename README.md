# Punk Website Clone

A fully responsive static Single Page Application (SPA) built with **HTML and CSS**, inspired by the CryptoPunks NFT website design.

This project was developed from a Figma design file and focuses on layout precision, responsiveness, and modern CSS techniques.

---

## 🚀 Overview

This is a beginner front-end project built to strengthen my understanding of:

- Semantic HTML structure  
- Mobile-first responsive design  
- Modern CSS layout techniques  
- Translating Figma designs into real-world code  

The goal was to replicate the design as closely as possible while maintaining a clean and scalable structure.

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- Flexbox
- CSS Grid
- Media Queries
- `clamp()` for fluid responsiveness
- Mobile-first workflow

> No JavaScript was used in this project.

---

## 📱 Responsiveness

The project was developed using a **mobile-first approach** and is fully responsive across:

- Mobile devices  
- Tablets  
- Desktop screens  

Layout behavior adapts using media queries and fluid sizing techniques to maintain visual consistency across screen sizes.

---

## ✨ Features

- Responsive hamburger menu  
- Smooth scrolling  
- CSS-based carousel animation  
- Hover effects for interactive elements  
- Structured single-page layout  

---

## 🎨 Design Implementation

- Built directly from a Figma design file  
- Not perfectly pixel-for-pixel, but visually very close  
- Minor spacing and structural adjustments were made for responsiveness and cleaner CSS architecture  
- No additional features beyond the original design were added
